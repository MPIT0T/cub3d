
# cube3d

A small RayCaster with the C graphic library : MiniLibX


## Installation

Install cube3d with makefile :

```bash
  git clone https://github.com/MPIT0T/cube3d.git
  cd cube3d
  make
```

Run it :
```bash
./cube3d maps/<desired_map>.ber
```
