
# cub3d

A small RayCaster with the C graphic library : MiniLibX


## Installation

Install cub3d with makefile :

```bash
  git clone https://github.com/MPIT0T/cub3d.git
  cd cub3d
  make
```

Run it :
```bash
./cub3d maps/<desired_map>.cub
```
